建立交通禮讓態度很重要嗎?
據統計台灣一年造成傷亡的交通意外事故將近25萬件。
請同學撰寫一JAVA類別 TrafficSimulation，根據下列數據，模擬車禍傷亡統計結果。
假設駕駛人分為兩類：禮讓駕駛，搶快駕駛兩類。每類型各５００人。
當兩位駕駛會車時，發生車禍的機率假設如下表：

 	禮讓	搶快
禮讓	0.01	0.05
搶快	0.05	0.95
當發生車禍時，造成死亡的機率假設如下表：

 	禮讓	搶快
禮讓	0	1
搶快	1	1
1： 雙方死亡
撰寫JAVA程式碼，模擬一百萬件隨機會車時（也就是從所剩下的駕駛人當中任選兩位駕駛人），
根據其發生車禍與造成傷亡的機率，統計下列資訊：
總共發生幾次交通事故？
其中造成死亡的交通事故有幾件？
其中禮讓駕駛人死亡幾位？
其中搶快駕駛人死亡幾位？

提示：TrafficSimulation 物件導向分析

1. 駕駛人（Driver）：具有類型屬性（禮讓、搶快）。

2. 會車事件（Encounter）：兩位駕駛隨機組合，依照表格機率決定是否發生事故與死亡。

3. 模擬控制器（TrafficSimulation）：負責重複執行多次會車、統計結果。


測試類別: 
import java.util.*;

public class Main {

     public static void main(String[] args) {
        TrafficSimulation sim = new TrafficSimulation();
        sim.runSimulation(1_000_000);
        sim.report();
    }
}
